PALADONE LICENSE LOGOS
======================

Drop official brand SVG logos here when Paladone supplies them.
Filename must match exactly:

  lego.svg            mario.svg          minecraft.svg
  starwars.svg        hp.svg             marvel.svg
  kpop.svg            sanrio.svg         ps.svg
  xbox.svg            astro.svg          fortnite.svg
  onepiece.svg        stitch.svg         disney.svg
  nbc.svg             beetlejuice.svg    it.svg
  bluey.svg           carebears.svg      minions.svg
  toystory.svg        pooh.svg           superman.svg

When a logo file exists at this path, the typographic
fallback is automatically hidden. PNGs work too — just
change the file extension in the index.html src attribute.
